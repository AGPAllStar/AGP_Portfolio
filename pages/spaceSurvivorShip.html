Game Description

This game was created with the 2D Unity Template.

This is a spaceship game where you must survive. You’ll need to destroy as many asteroids as possible to score points. If you get hit 3 times, you lose. You have 2 weapons in your arsenal: missiles and bombs.

Tasks performed

System Design

I used the Unity physics system, adjusting the gravity and the items collisions. The game’s logic, what is shown on screen and the player’s controls have become out of sync.

Game Design

I adjusted the asteroid spawn to have always asteroids around. Spaceship speed and inertia force player to be careful not to crash to asteroids.

Performance

Asteroids which are too far from player are erased from world so as not to use resources unnecessarily.

2D Art

I've created almost all the 2D sprites for the game. I used a pixel art style due to its simplicity.
